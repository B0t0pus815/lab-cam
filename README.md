# lab-cam
lab's cam — 实验室摄影机 App（Android）

打开即看实验室即时画面（直播），并可回放录影片段。

## 下载安装（手机）

手机浏览器打开下面这个网址，下载最新版 APK：

**👉 https://github.com/B0t0pus815/lab-cam/raw/main/app-release.apk**

> 这个网址永远指向最新版。之后有更新，重开同一网址下载、覆盖安装即可。

第一次安装会被问「是否允许安装未知应用程式」，允许即可。

## 使用

1. 打开 App，预设有两个分页：**直播** / **回放**。
2. 点 **設定主機**，填入主机位址后套用：
   - 和主机同一个区网时：填主机内网 IP（例如 `10.10.10.191`）。
   - 在外面 / 透过 Tailscale 时：填 `ros2project`（需手机与主机都在同一 tailnet）。
3. **直播**走 RTSP（`rtsp://主机:8554/lab`）；**回放**从主机的唯读录影服务（`主机:8780`）列出片段并播放。

## 备注
- 直播用 RTSP、回放用 HTTP，主机端防火墙需放行对应埠（8554 / 8780）。
- App 仅做即时观看与录影回放，不录影、不改主机上任何服务。
